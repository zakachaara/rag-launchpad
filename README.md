# Rag LaunchPad
This toolkit enables you to launch as rag system as you want by defining your configuration in the `frontend`. 

The RAG system is handled by a Ollama Server. 

## Test it:

1. *Automated Installation Script*
```bash
curl -fsSL https://raw.githubusercontent.com/YOUR_USERNAME/rag-launchpad/main/install.sh | sh
```

2. Or *Clone the Repo and run*

```bash
./start.sh --setup

sleep 10 # wait

./start.sh

```
